TRAVELOOP 🌍✈️

A modern AI-powered travel planning platform built for seamless trip organization, collaboration, budgeting, and exploration.

TRAVELOOP helps travelers plan trips smarter with interactive itineraries, live travel tools, maps, budgeting utilities, travel journals, and collaborative planning — all inside one beautiful futuristic dashboard.

⚙️ Installation & Setup

1️⃣ Clone Repository

```bash
git clone <repository-url>
cd TRAVELOOP
```

---

2️⃣ Install Dependencies

Frontend

```bash
cd client
npm install --legacy-peer-deps
```

Backend

```bash
cd ../server
npm install --legacy-peer-deps
```

---

▶️ Run Development Server

Start Frontend

```bash
cd client
npm run dev
```

Start Backend

```bash
cd server
npm run dev
```

---

🐳 Run with Docker

```bash
docker compose up
```

---

🌐 Environment Variables

Create a `.env` file in both client and server directories if required.

Example:

```env
PORT=3000
DATABASE_URL=your_database_url
REDIS_URL=your_redis_url
JWT_SECRET=your_secret_key
```
